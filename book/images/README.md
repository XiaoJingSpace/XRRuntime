# 图片目录说明

本目录用于存放文档中需要的截图和图片。

## 目录结构

```
book/images/
├── chapter02/          # 第2章截图
├── chapter03/          # 第3章截图
├── chapter04/          # 第4章截图
├── chapter05/          # 第5章截图
├── chapter15/          # 第15章截图
├── chapter16/          # 第16章截图
├── chapter17/          # 第17章截图
└── chapter19/          # 第19章截图
```

## 截图命名规范

格式：`chapterXX_sectionXX_description.png`

示例：
- `chapter02_section01_android_studio_install.png`
- `chapter04_section01_gradle_sync.png`
- `chapter15_section01_jnilibs_structure.png`

## 截图要求

1. **分辨率**: 建议 1920x1080 或更高
2. **格式**: PNG（推荐）或 JPG
3. **标注**: 重要区域应使用箭头或高亮标注
4. **文件大小**: 建议每个截图不超过 2MB
