Enumerator

The user is being tracked and the value has been updated.

kFoveatedEyeGazeStateExtrapolated 

The user is not being tracked and the value has been extrapolated from historical data.

kFoveatedEyeGazeStateLastKnown 

The user is not being tracked and the value is a repeat of the last tracked value.
